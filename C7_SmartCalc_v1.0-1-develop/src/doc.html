Установка Калькулятора

Для установки калькулятора наберите команду в терминалеmake install.

Архивация Калькулятора

Для архивации команду make dvst.

Вычисление выражения

Для вычисления выражения введите в поле выражение и нажмите на кнопку Calculate.

Калькулятор может производить основные арифметические действия (сложение, вычитание, умножение, деление), тригонометрические (синус, косинус, тангенс, арксинус, арккосинус, арктангенс), а также возведение в степень, остаток от деления, квадратный корень и вычислять натуральный и десятичный логарифмы.

У пользователя есть возможность ввода до 255 символов

На вход программы могут подаваться как целые числа, так и вещественные числа, записанные через точку.

Есть возможность отобразить график функции. Для построения графиков функции необходимо дополнительно указывать отображаемые область определения и область значения. Область определения и область значения функций ограничиваются числами от -1000000 до 1000000. Для построения графика необходимо нажать на кнопку graph.

Реализована возможность подстановки значения переменной X. Для этого необходимо ввести выражение с X и нажать на кнопку Calculate, а далее в открывшемся окне ввести значение X. После этого в основном окне появится значение введенного выражения.

Для ввода нового выражения необходимо нажать на кнопку Try again.
